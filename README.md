💡 Анти-ИИ Свет
Честный офлайн-измеритель освещённости в люксах. Социальный проект. Без облака. Без трекеров.
Веб-приложение для измерения освещённости через камеру смартфона. Разработано как инструмент цифровой гигиены: показывает реальные параметры света, помогает калибровать замеры и предупреждает о алгоритмических искажениях (HDR, авто-экспозиция, тонмаппинг).

🧮 Как работает алгоритм
Захват: Камера (facingMode: 'environment') передаёт поток в скрытый <canvas> (160×120)
ROI-анализ: Берётся центральная зона getImageData(40, 30, 80, 60)
Яркость: Считается по стандарту Rec.709
Y = 0.2126·R + 0.7152·G + 0.0722·B
Гамма-коррекция: Компенсация нелинейности мобильного ISP
linear = (Y / 255) ^ 1.8
Перевод в люксы: Lux = linear × 1200 × K
K — калибровочный коэффициент (по умолчанию 1.0)
Качество замера: Учитывает долю пересвеченных/затемнённых пикселей и дисперсию яркости за последние 30 кадров
Сглаживание: Экспоненциальный фильтр smoothedLux = prev × 0.9 + cur × 0.1
⚠️ Ограничения и отказ от ответственности
Приложение не является метрологическим прибором и не заменяет профессиональный люксметр.
Браузерный API не даёт доступ к RAW-данным матрицы. Результат зависит от встроенного ISP, автоэкспозиции и tone-mapping телефона.
Для максимальной точности отключите HDR, Night Mode и «улучшение фото» в настройках камеры устройства.
Показатели носят ориентировочный характер и предназначены для бытовой цифровой гигиены.
